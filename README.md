# Client Connect (Plain PHP)

This is a framework-free PHP project scaffold for the Client Connect assessment.

## Tech
- PHP (no framework)
- MySQL (planned)
- JavaScript (planned)

## Initial structure
- `app/` - MVC and core application classes
- `bootstrap/` - startup/bootstrap files
- `config/` - configuration files
- `database/` - schema and seed scripts
- `public/` - web root (front controller)
- `resources/` - static source assets (css/js)
- `routes/` - route definitions
- `storage/` - logs, cache, uploads
- `tests/` - test scripts
